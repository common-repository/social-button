== Description ==
Social button is a simple plugin for showing your social profile button in any page, post or widget. You can use it by it's 3 different sizes. 

Features:
1) Support on post, page and widget
2) very light weight.
3) Nice design
4) Easy to embed.
5) Many icons available. 
6) 3 diffrent sizes avilable.




== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Write [social_btn link="http://twitter.com/yourname" name="twitter" size="medium"] in page, post or widget to showing twitter icon.
4. change then name to your desired social icon , link to your desired link/website and size for changing size of social buttons 
. you can write there "small" instead of medium for showing icons to small. You can also avoid size for getting default size. Then it will 
be like this : [social_btn link="http://twitter.com/yourname" name="twitter"]
5. If you only write [social_btn] , it will show facebook icon... Because it's default icon is facebook.
6. Here, you can use following name with following social button:
*facebook
*twitter
*google
*skype
*drrrible
*behance
*flickr
*youtube
*linkedin
*stackoverflow
*pinterest
*reddit
*plus-google
*tumblr

7. You can also use any name from font-awesome directory for more button on social icon..you can search your icon from here....http://fontawesome.io/icons/

Thats it.
